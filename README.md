一款极简、隐私优先的 Android 习惯追踪应用。专注于「记录」本身，用温暖的设计和流畅的体验，帮你轻松养成好习惯。

## ✨ 主要特性
- **极简体验**：摒弃繁杂功能，一键创建打卡项目，直观回顾历史记录。
- **数据安全**：采用离线优先策略，所有数据本地存储，不上传云端，隐私有保障。
- **温暖设计**：莫兰迪色系搭配流畅动画，提供沉浸式的视觉体验。
- **可视化分析**：自定义图表组件，清晰展示你的坚持成果。

## 🔧 技术栈
- **架构模式**：MVVM + Jetpack (Room, ViewModel, LiveData)
- **开发语言**：Kotlin
- **UI 框架**：Material Design 3
- **数据存储**：Room 本地数据库
- **图表库**：MPAndroidChart

## 📦 安装方式
1.  克隆项目到本地：
    ```bash
    git clone https://github.com/pnmgg/PunchClock.git
    ```
2.  用 Android Studio 打开项目，等待依赖下载完成。
3.  连接设备或启动模拟器，点击「Run」按钮即可运行。
